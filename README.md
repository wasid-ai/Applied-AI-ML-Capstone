# Adult Census Income Prediction using Machine Learning

## Project Overview

This project focuses on predicting whether an individual's annual income exceeds **$50K** using the Adult Census Income Dataset. The project includes data preprocessing, exploratory data analysis (EDA), machine learning model development, ensemble learning, hyperparameter tuning, and structured JSON-based model analysis.

---

## Dataset

* **Dataset:** Adult Census Income Dataset
* **Target Variable:** `income`
* **Task:** Binary Classification
* **Language:** Python
